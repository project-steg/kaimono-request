# Stegチュートリアル: 「買ってきてほしいものリスト」Webアプリを作ろう！

最終更新日: 2020年2月8日

## 概要

当レポジトリは、学生ITエンジニア団体Stegにてメンバーのスキル向上のために作成した「Stegチュートリアル」で使用するレポジトリです。  
このチュートリアルでは、「買ってきてほしいものリスト」Webアプリを作成することを通じて、フロントエンド・バックエンド・インフラへの理解を深めてもらうことを目的としています。
